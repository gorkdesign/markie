# BackstopJS

## Instructions for Set Up for First Time (new projects)
- Install backstopjs globally
-- `npm install -g backstopjs`
- Initialise backstop
-- `backstop init`
- Set up the tests
-- Edit backstop.json
- Run the references
-- backstop reference
- Run the tests
-- backstop test
- Write more tests

# Something not working?
You probably need a more recent version of node
- nvm install 8
